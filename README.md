# Заявка на семинар (React + Vite + TypeScript)

Простой проект для отправки заявки на семинар.

## Что умеет
- форма (имя, email, телефон, организация, сообщение)
- отправка на `mailto:` (через `window.location.href`)
- базовая валидация (обязательные поля)

> Важно: отправка через `mailto:` не требует бэкенда, но пользовательский почтовый клиент должен быть настроен.

## Установка и запуск
```bash
npm install
npm run dev
```

## Сборка
```bash
npm run build
```

## Переменные
Адрес получателя указан в `src/constants.ts`.
